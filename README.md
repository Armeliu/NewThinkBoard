# QuizDuel

QuizDuel is a MERN-based quiz battle platform. This repository contains the initial monorepo scaffolding for the client and server applications.

## Local setup

1. Ensure MongoDB is running locally.
2. Copy the example environment file for the server:

```bash
cp server/.env.example server/.env
```

3. Install dependencies:

```bash
npm install
```

## Run in development

```bash
npm run dev
```

- Client: http://localhost:5173
- Server: http://localhost:4000

## Run tests

```bash
npm run test
```
